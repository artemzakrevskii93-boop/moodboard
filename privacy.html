Privacy Policy

AI Moodboard Photoshop Plugin · Last updated: March 2025

Short version: This plugin does not collect, store, or transmit any personal data to our servers. All credentials are kept locally on your device.

1. About This Application

AI Moodboard is a personal plugin for Adobe Photoshop that helps concept artists create visual reference boards. It uses the Pinterest API to search for reference images based on user-provided prompts.

This tool is developed and maintained by Artem Zakrevskii as a personal productivity tool for individual artists.

2. Data We Do NOT Collect

We do not collect, process, or store:

  • Names, email addresses, or any personal identifiers
  • Usage statistics or analytics
  • Search queries or prompts entered by users
  • Images retrieved from Pinterest
  • Device information or IP addresses

3. Local Storage of Credentials

The plugin allows users to enter a Backend URL to connect to a local server. This value is stored exclusively in your browser's localStorage — on your own device — and is never transmitted to any third party or external server controlled by this application.

4. Third-Party Services

The plugin connects to the following external services on the user's behalf:

  • Pinterest API — used to search for reference images. Pinterest's own Privacy Policy applies to all interactions with their API.
  • Groq API (optional) — used to generate search queries from user prompts. Groq's own Privacy Policy applies.
  • Anthropic API (optional alternative) — Anthropic's Privacy Policy applies.

All API keys and tokens for these services are provided by the user, stored locally on their device, and used solely to perform the functions described above.

5. Pinterest API Usage

This application uses the Pinterest API to perform read-only image searches (pins:read, boards:read scopes). The plugin does not create, modify, or delete any Pinterest content. It does not access Pinterest user account data, boards, or followers.

6. Children's Privacy

This plugin is intended for use by professional and hobbyist artists. It is not directed at children under the age of 13, and we do not knowingly collect any information from children.

7. Changes to This Policy

If this policy is updated, the revised version will be published at this URL with a new "Last updated" date. Continued use of the plugin after changes constitutes acceptance of the updated policy.
