AI Moodboard

A Photoshop UXP plugin that generates reference moodboards for concept artists using AI-powered image search.

How it works

1
Describe your subject Type a description in any language — character, environment, prop or building. The AI understands the style, mood and theme.
2
AI generates targeted search queries Each grid cell has a specific role (silhouette, pose, face, materials, weapons, color palette…). A language model creates a precise Pinterest search query for every cell.
3
Pinterest search per cell The backend searches Pinterest for each cell independently, returning up to 10 candidates. The best matching image is selected using keyword scoring.
4
Moodboard created in Photoshop Images are downloaded and placed as Smart Object layers inside a named group — ready to use in your PSD.

Features

⚔️
4 scene types Character, Environment, Prop, Building — each with professional CGI slot structure.
⚙️
6 layout sizes 6 to 20 images. Three layout variants (A/B/C) per size.
⬡
No duplicates Automatic deduplication — every cell gets a unique image.
↺
Regen & Undo Not happy? Regenerate with one click or revert to previous set.

Data & Privacy

  • The plugin does not collect any personal data.
  • API keys are stored locally in the browser (localStorage) and never sent to our servers.
  • Image search is performed via Pinterest API using your own access token.
  • This tool is intended for personal and professional use by individual artists.
